APseudocode is the non executable form of code whether it is of python programming language or whichever. Pseudocode is written to analyse how the code will be written and how the whole process will look like. The reason why pseudocode is written is same as writing algorithm and flowchart.

It is not mandatory to write pseudocode before writing the code but it helps to understand and write code smoothly if pseudocode is already written.
