Privacy Policy for Cover Letter Writer Extension

Effective Date: August 14, 2024

Introduction

Cover Letter Writer ("we," "our," or "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and protect your information when you use our Chrome extension.

Information We Collect

CV/Resume Data

  • We temporarily process your CV/resume files (PDF, DOC, DOCX, TXT) to generate cover letters
  • This data is processed through our secure Railway-hosted API
  • We do not permanently store your CV content

Job Description Data

  • We extract job descriptions from web pages you visit
  • This data is used solely for cover letter generation
  • We do not store job descriptions permanently

Usage Information

  • Basic extension usage statistics (features used, error reports)
  • No personally identifiable information is collected

How We Use Your Information

Cover Letter Generation

  • Your CV data is processed by OpenAI's API to generate personalized cover letters
  • Job descriptions are analyzed to match your experience with job requirements
  • All processing is done securely through our Railway infrastructure

Service Improvement

  • Usage analytics help us improve the extension's functionality
  • Error reports help us fix bugs and improve performance

Data Security

Encryption

  • All data transmission uses HTTPS encryption
  • API calls are secured with industry-standard protocols

Storage

  • CV data is processed in memory and not permanently stored
  • No personal information is retained on our servers
  • Temporary processing data is automatically cleared

Third-Party Services

  • We use OpenAI's API for AI-powered content generation
  • OpenAI's privacy policy applies to data processed by their services
  • Railway hosts our API infrastructure securely

Data Retention

  • CV/Resume Data: Not stored permanently
  • Job Descriptions: Not stored permanently
  • Usage Analytics: Retained for service improvement (anonymized)

Your Rights

Access and Control

  • You can uninstall the extension at any time
  • You control what CV data you upload
  • You can choose which job sites to use the extension on

Data Deletion

  • Since we don't permanently store your data, no deletion is needed
  • Uninstalling the extension removes all local data

Children's Privacy

This extension is not intended for children under 13. We do not knowingly collect personal information from children under 13.

Changes to This Policy

We may update this Privacy Policy from time to time. We will notify users of any material changes through the extension or our website.

Compliance
